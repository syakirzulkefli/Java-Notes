# 5 Interesting Facts About Java

## Overview

- In this lesson, you’ll learn 5 fun and important facts about **Java**—its origins, editions, reach, and career potential.
- These facts help you appreciate Java’s history, popularity, and opportunities.

---

## 1. Java’s Origins

- Java was developed by **James Gosling** in **1995** at **Sun Microsystems**.
- It was originally named:
  - `Oak` – after an oak tree outside Gosling’s office.
  - Then renamed to `Green`.
  - Finally renamed to **Java**, inspired by **Java coffee** ☕.
- Sun Microsystems was later **acquired by Oracle** in **2010**.

---

## 2. Java Editions

Java comes in **four editions**, each designed for different kinds of applications:

- **Java SE (Standard Edition)**

  - Core Java platform used in this course.
  - Includes essential libraries every Java developer should know.

- **Java EE (Enterprise Edition)**

  - Used for building **large-scale, distributed, and multi-tier** applications.
  - Built on top of Java SE with advanced features (e.g., fault tolerance, scalability).

- **Java ME (Micro Edition)**

  - Subset of Java SE tailored for **mobile devices**.
  - Includes libraries specific to embedded and mobile systems.

- **Java Card**
  - Designed for **smart cards** (used in banking, ID, etc.).

---

## 3. Java’s Reach

- As of Java SE 12 (released **March 2019**):
  - Java is used by over **9 million developers** worldwide.
  - Java runs on:
    - ~**3 billion mobile phones**
    - ~**120 million TV sets**
    - **Every Blu-ray player**

---

## 4. Developer Salary

- According to _Indeed.com_:
  - The **average salary** of a Java developer in the **United States** is **just over $100,000/year**.

---

## 5. Java is Everywhere

- Java is a **ubiquitous language** found in mobile devices, web apps, enterprise software, and more.
- This ubiquity leads to:
  - **High demand** for Java developers
  - **Wide range of job opportunities**

---

## Summary

- Java has a rich history and diverse ecosystem.
- It supports multiple editions for different platforms.
- It powers billions of devices globally.
- Java developers are well-compensated and in demand.
- Mastering Java opens the door to many **professional programming opportunities**.
