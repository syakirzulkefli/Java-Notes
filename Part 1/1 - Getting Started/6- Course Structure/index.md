# Course Structure

## Overview

- This course is part of a **4-part complete Java series**.
- Each part is around **3–4 hours long** — ideal for finishing in **a day or two**.
- The course is designed to progressively build your **Java skills**, from fundamentals to advanced topics.

---

## Part 1: Programming Fundamentals with Java

- Learn the **basics of programming** using Java.
- Topics include:

  - **Type System**
    - Work with numbers, strings, booleans, and arrays.
    - Build a **mortgage calculator** as your first Java project.
  - **Control Flow Statements**
    - Learn conditionals and loops to build **algorithms**.
    - Add **data validation** to your mortgage calculator.

- By the end of Part 1:
  - You’ll be able to write clean, structured Java code.
  - You’ll know how to build and improve simple apps like calculators.

---

## Clean Coding

- Learn **how to write clean and expressive code**.
- Techniques used by **professional developers**:
  - Naming conventions
  - Code organization
  - Maintainability
- Goal: Move from **average** to **outstanding** programmer.

---

## Debugging & Packaging

- Learn to:
  - **Find and fix errors** in your Java programs.
  - **Package applications** for deployment so others can use them.

---

## Part 2: Object-Oriented Programming (OOP)

- Master the **object-oriented programming** paradigm.
- Critical for all Java applications — desktop, web, or mobile.
- Without OOP knowledge, you’ll constantly hit roadblocks.

---

## Part 3: Core Java APIs

- Learn how to use **Java’s standard library (APIs)**.
- Topics include:
  - Commonly-used classes and utilities
  - Built-in Java tools to speed up development

---

## Part 4: Advanced Java Features

- Deep dive into **advanced Java capabilities**:
  - **Streams**
  - **Multithreading**
  - **Database programming**
  - And more...

---

## Final Note

- This course gives you a **solid foundation** to start building Java applications.
- Join the full journey to master **one of the most popular languages** powering millions of apps and websites.
