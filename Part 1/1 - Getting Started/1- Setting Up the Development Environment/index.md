# Java Development Environment Setup

## Overview

- This Java tutorial guides you through downloading and installing necessary tools to build Java applications.

## Download and Install JDK

- **Search for JDK download:**  
  JDK (Java Development Kit) is a software development environment for building Java applications. It includes:

  - A compiler
  - Reusable code libraries
  - A Java runtime environment

- **Steps:**

  - Visit [Oracle’s Java SE page](https://www.oracle.com/java/technologies/javase-downloads.html).
  - Click on the **Java SE** section.
  - Choose the JDK for your platform:
    - **Linux**
    - **macOS**
    - **Windows**

- **Example (macOS):**
  - Download the **DMG file**.
  - Accept the license agreement before downloading.
  - Open the downloaded DMG and double-click the package.
  - Follow the installation wizard (click **Continue**, then **Install**).
  - Enter your computer’s password if prompted.
  - Once the installation is complete, move the installer to **Trash**.

## Install a Code Editor

- **Popular code editors:**

  - **NetBeans**
  - **Eclipse**
  - **IntelliJ IDEA**

- **Editor used in this course:**
  - IntelliJ IDEA (Community Edition).
  - Download the free Community Edition from the official site.
  - Drag the IntelliJ application into the **Applications folder** after downloading.

## Conclusion

- All necessary tools are now installed to build Java applications.
