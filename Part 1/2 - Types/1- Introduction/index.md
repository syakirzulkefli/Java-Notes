# Introduction

## Overview

- In this section, you'll explore the **fundamentals of programming in Java**.
- These fundamentals will form the foundation for your first project: a **mortgage calculator**.

---

## What You’ll Learn

- **Variables and Constants**

  - Understand how to store and label data in your programs.

- **Primitive and Reference Types**

  - Learn the difference between basic types (like `int`, `double`) and complex types (like `String`, `arrays`).

- **Casting (Type Conversion)**

  - Learn how to convert between different data types safely and effectively.

- **Working with Common Data Types**

  - **Numbers** – arithmetic and operations
  - **Strings** – text manipulation
  - **Arrays** – storing and managing multiple values

- **Reading Input from the User**
  - Get data dynamically from user interaction

---

## Final Project

- After learning these concepts, you'll build a **Mortgage Calculator** on your own.
- This project will **apply everything** you’ve learned in this section.
