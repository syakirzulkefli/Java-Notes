# Study Notes for 16- Summary

## Overview

In this section, you learned how to **control the flow of execution** in your Java programs.

---

## Key Topics Covered

### 🔹 Comparison Operators

- Used to **compare primitive values**.
- Examples: `==`, `!=`, `<`, `<=`, `>`, `>=`.

---

### 🔹 Logical Operators

- Combine multiple Boolean expressions.
- Operators:
  - `&&` – Logical AND
  - `||` – Logical OR
  - `!` – Logical NOT
- Used for implementing **real-world decision rules**.

---

### 🔹 Conditional Statements

- **`if` statements** – Run code based on condition.
- **`switch` statements** – Cleanly handle multiple possible values.

---

### 🔹 Looping Statements

- Used for **executing code repeatedly**.
- Four types:
  - `for` loop – Best when count is known.
  - `while` loop – Best when count is not known.
  - `do...while` loop – Runs **at least once**.
  - `for-each` loop – Ideal for **iterating over arrays/collections**.

---

### 🔹 Loop Control

- `break` – Exits the loop early.
- `continue` – Skips current iteration and continues the loop.

---

- Make decisions in your programs.
- Handle loops and repetitions.
- Build real-world logic in Java apps.
