# Introduction

## Overview

In this section, you'll learn how to **control the flow** of your Java programs — deciding **what code runs** and **when** it runs.

---

## What You'll Learn

### ✅ Comparison Operators

- Used for **comparing values**.
- Examples:
  - `==` (equal to)
  - `!=` (not equal to)
  - `>` (greater than)
  - `<` (less than)
  - `>=`, `<=`

### ✅ Logical Operators

- Combine multiple conditions using:
  - `&&` (logical AND)
  - `||` (logical OR)
  - `!` (logical NOT)

> 🧠 These help implement **real-world decision rules** in code.

---

## Control Flow Statements

1. **Conditional Statements**

   - `if`, `else if`, `else`
   - Used for **making decisions**

2. **Loops**

   - `for`, `while`, `do-while`
   - Used for **repeating code**

3. **Enhancing the Mortgage Calculator**
   - Add **error handling**
   - Validate user input (e.g., disallow negative values or invalid types)
